**Step 1: Install shadcn Tabs component**

Run `npx shadcn@latest add tabs` from the project root. This installs `@radix-ui/react-tabs` as a dependency and generates `src/components/ui/tabs.tsx` with exports: `Tabs`, `TabsList`, `TabsTrigger`, `TabsContent`.

Verify the file exists at `src/components/ui/tabs.tsx` after installation.

**Step 2: Create the server component page at `src/app/my-clubs/[id]/page.tsx`**

This is a Server Component (no "use client" directive). It:

1. Accepts `params: Promise<{ id: string }>` and `searchParams: Promise<{ tab?: string }>` — Next.js 16 requires awaiting both.
2. Creates a server Supabase client via `createClient()` from `@/lib/supabase/server`.
3. Gets the authenticated user via `supabase.auth.getUser()`. If no user, `redirect("/")`.
4. Runs a `Promise.all` with three parallel queries:
   - Fetch the club: `supabase.from("clubs").select("*").eq("id", id).single()`
   - Fetch the user's membership: `supabase.from("club_members").select("role").eq("club_id", id).eq("user_id", user.id).single()`
   - Fetch member count: `supabase.from("club_members").select("*", { count: "exact", head: true }).eq("club_id", id)`
5. If `!club || !membership`, call `notFound()` (from `next/navigation`).
6. Cast `membership.role as "owner" | "organizer"`.
7. Conditionally fetch pending invites count ONLY if `role === "owner"`:
   - `supabase.from("club_invitations").select("*", { count: "exact", head: true }).eq("club_id", id).eq("status", "pending")`
   - Pass `pendingInvitesCount` as number for owners, `null` for organizers.
8. Render `<ClubDashboard>` passing props: `club`, `role`, `memberCount`, `pendingInvitesCount`, `initialTab` (from searchParams tab or "overview").

Import `notFound` and `redirect` from `next/navigation`. Import `createClient` from `@/lib/supabase/server`. Import `ClubDashboard` from `@/components/clubs/ClubDashboard`. Import `Club` type from `@/types`.

**Step 3: Create the client dashboard shell at `src/components/clubs/ClubDashboard.tsx`**

This is a `"use client"` component. Structure:

1. Define `ClubDashboardProps` interface:
   ```typescript
   interface ClubDashboardProps {
     club: Club;
     role: "owner" | "organizer";
     memberCount: number;
     pendingInvitesCount: number | null;
     initialTab: string;
   }
   ```

2. Define `VALID_TABS = ["overview", "events", "members", "settings"] as const` and `type TabValue = (typeof VALID_TABS)[number]`.

3. Create an inner component `ClubDashboardInner` that uses `useSearchParams()`, `useRouter()`, and `usePathname()` from `next/navigation`:
   - Read current tab: `searchParams.get("tab") ?? initialTab`
   - Validate against `VALID_TABS`, default to `"overview"` if invalid
   - `handleTabChange(value: string)`: create new `URLSearchParams`, set "tab", call `router.replace(`${pathname}?${params.toString()}`)` — use `replace` not `push` to avoid history pollution

4. Render a page header section above the tabs:
   - Show club name as `<h1>` with Building2 icon (from lucide-react)
   - Show role badge using `<Badge>` component — capitalize the role text
   - Style with Tailwind: `container mx-auto px-4 py-8`

5. Render `<Tabs value={validTab} onValueChange={handleTabChange}>`:
   - `<TabsList>` with four `<TabsTrigger>` entries: "overview" (label: "Overview"), "events" (label: "Events"), "members" (label: "Members"), "settings" (label: "Settings")
   - `<TabsContent value="overview">`: render `<ClubOverviewTab>` with props `club`, `memberCount`, `pendingInvitesCount`, `role`
   - `<TabsContent value="events">`: render a placeholder div with text "Events tab content will appear here." and a muted style. This stub will be replaced in Plan 02.
   - `<TabsContent value="members">`: render a placeholder div: "Members management coming soon."
   - `<TabsContent value="settings">`: render a placeholder div: "Club settings coming soon."

6. Export `ClubDashboard` as a wrapper that renders `<Suspense>` around `<ClubDashboardInner>` — required because `useSearchParams()` needs a Suspense boundary in Next.js App Router.

Import `Suspense` from "react". Import `useSearchParams`, `useRouter`, `usePathname` from "next/navigation". Import `Tabs`, `TabsList`, `TabsTrigger`, `TabsContent` from `@/components/ui/tabs`. Import `Badge` from `@/components/ui/badge`. Import `Building2` from "lucide-react". Import `Club` from `@/types`. Import `ClubOverviewTab` from `@/components/clubs/ClubOverviewTab`.
