  Build all club-related API routes from scratch using patterns from 01-RESEARCH.md.

  **GET/PATCH /api/clubs/[id]/route.ts:**
  - GET: Fetch club by id from `clubs` table. In parallel, fetch follower count using `select("*", { count: "exact", head: true })` on `club_followers`. Return `{ club, followerCount }`. No auth required (public page).
  - PATCH: Auth required. Check `club_members` for user's membership with `role = 'owner'`. If not owner, return 403. Accept body fields: `name`, `description`, `category`, `instagram_handle`, `logo_url`. Trim strings, set empty strings to null. Update with `.update().eq("id", clubId).select("*").single()`. Return `{ club }`.
  - Use `const { id: clubId } = await params;` (Next.js 16 async params).
  - Return generic error messages, never raw Supabase errors.
  - Define `interface RouteParams { params: Promise<{ id: string }> }`.

  **GET /api/clubs/[id]/events/route.ts:**
  - Auth not required. Fetch approved events for club: `.from("events").select("*, club:clubs(id, name, logo_url)").eq("club_id", clubId).eq("status", "approved").order("event_date", { ascending: true })`.
  - Return `{ events }`.

  **GET/POST/DELETE /api/clubs/[id]/follow/route.ts:**
  - GET: Auth required. Check if current user follows this club: `.from("club_followers").select("id").eq("user_id", user.id).eq("club_id", clubId).maybeSingle()`. Return `{ following: boolean }`.
  - POST: Auth required. Upsert into `club_followers` with `onConflict: "user_id,club_id"` and `ignoreDuplicates: true`. Return 201.
  - DELETE: Auth required. Delete from `club_followers` where `user_id = user.id` and `club_id = clubId`. Return 200.

  **GET /api/my-clubs/route.ts:**
  - Create new file at `src/app/api/my-clubs/route.ts`.
  - Auth required. Fetch user's club memberships: `.from("club_members").select("id, role, created_at, club:clubs(id, name, description, logo_url, category)")`.eq("user_id", user.id)`.
  - For each club, fetch member count in parallel using `Promise.all`: `.from("club_members").select("*", { count: "exact", head: true }).eq("club_id", club.id)`.
  - Return `{ clubs: [{ id, role, created_at, club: {...}, memberCount }] }`.

  **POST /api/clubs/logo/route.ts:**
  - Auth required. Accept FormData with `file` field. Validate file type (image/jpeg, image/png, image/webp) and size (max 2MB). Upload to Supabase Storage bucket `club-logos` with path `{clubId}/{timestamp}.{ext}`. Return `{ url }` with the public URL.
  - Check that user is owner of the club (club_id from form data).
