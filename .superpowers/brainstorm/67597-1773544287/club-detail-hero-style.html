A: Event-Style Full Hero

Mirrors your event detail page exactly — full-bleed banner image, club logo + info overlay at the bottom, gradient fade. The club's banner/cover image fills the hero. If no banner exists, fall back to a gradient with the club's category color.

B: Profile-Centered Hero

Club-specific treatment — centered logo (circular, large), name, category badge, and quick stats (followers, events, members) all within the hero. Uses a colored gradient background tied to the club's category rather than requiring a banner image. More "profile page" feel.

C: Banner + Floating Card

Hybrid approach — shorter banner image at top with a floating profile card that overlaps the banner. The card contains the logo, name, category, follower count, and action buttons. Similar to GitHub org pages or Discord server profiles. Works well with or without a banner image.

Note: All options will include follow button, social links, bio, event tabs, and category below the hero. This question is just about the hero treatment itself.
