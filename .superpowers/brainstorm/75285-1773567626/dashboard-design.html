Key Design Decisions

🧭 Navigation
  • Collapsed icon sidebar (expands on hover)
  • Hero + tabs replace the old sidebar nav
  • "View Public Page" link for preview
  • Mobile: no sidebar, scrollable tabs
  • FAB for "Create Event" on mobile
📊 Overview Tab
  • Full hero (banner + overlapping logo + stats)
  • "Edit Banner" overlay on hover
  • 4 stat cards with trend indicators
  • Next Event spotlight card
  • Recent events table
  • Completion nudge (if incomplete)
📈 Analytics Tab
  • Follower growth line chart (30 days)
  • Engagement rate trend
  • Best-performing event type insights
  • Per-event breakdown table
  • Popular tags distribution
  • Peak activity times
⚙️ Settings Tab
  • Logo + banner upload
  • Name, description, category
  • Contact email (required)
  • 5 social links (IG, X, Discord, LinkedIn, Web)
  • Danger zone: transfer/delete
